Cài đặt cấu hình iOS

Nhấn nút dưới đây. iPhone sẽ hỏi “Cho phép tải cấu hình?” → chọn Cho phép. Sau đó vào Cài đặt để hoàn tất.

  • 1Mở lời nhắc “Cho phép” và đồng ý.
  • 2Vào Cài đặt (sẽ hiện mục Hồ sơ đã tải về ở đầu) → Cài đặt.

Đang gửi file tới iPhone… Nếu chưa thấy gì sau ~2 giây, mở Hồ sơ đã tải về trong Cài đặt.

Mẹo: Nếu “nhấn giữ” lâu trên nút, Safari có thể hiện menu tải về. Hãy chỉ chạm 1 lần để mở trình cài đặt cấu hình.
