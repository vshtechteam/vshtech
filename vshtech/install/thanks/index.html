Yêu cầu cài đặt đã được gửi tới iPhone

Hoàn tất cài đặt cấu hình vào Hồ sơ

Làm theo các bước dưới đây trong Cài đặt để hoàn tất cài đặt tệp .mobileconfig:

  • 1Mở Cài đặt và chạm mục Hồ sơ đã tải về (xuất hiện ở đầu).
  • 2Chọn hồ sơ cấu hình .mobileconfig → Cài đặt.
  • 3Nhập mã mật khẩu (nếu được yêu cầu) → Cài đặt để xác nhận.
Quay lại trang tải

Nếu chưa thấy mục “Hồ sơ đã tải về”, đợi vài giây hoặc bấm Gửi lại hồ sơ.
